================================================================================
                       TWO MAIN PROJECT GOALS
================================================================================

GOAL 1: BINARY CLASSIFICATION (Rock vs Mine)
Implemented in: make_prediction() function

INPUT: 60 SONAR frequency band values (0-1)
OUTPUT:
- Classification: "Rock" (safe) or "Mine" (alert)
- Confidence: Probability percentage (0-100%)
- Risk Level: SAFE / MODERATE / HIGH / CRITICAL
- Recommendation: Actionable guidance for users

EXAMPLE:
  Input: [0.02, 0.04, 0.05, ..., 60 values total]
  Output: {
    "object_type": "Rock",
    "confidence": 89.5,
    "risk_level": "SAFE",
    "recommendation": "Safe to proceed"
  }

---

GOAL 2: FEATURE IMPORTANCE EXPLANATION
Implemented in: get_risk_factors() function

OUTPUT: Top 10 SONAR frequency bands that distinguish rocks from mines
- Rank: Position in importance ranking
- Frequency Band: Band number (0-59)
- Importance Score: Raw importance value
- Percentage: Relative importance (0-100%)

EXAMPLE:
  Rank 1: Frequency Band 42, Importance: 0.45, Contribution: 15.2%
  Rank 2: Frequency Band 39, Importance: 0.38, Contribution: 12.8%
  Rank 3: Frequency Band 45, Importance: 0.35, Contribution: 11.9%
  ...

INSIGHT: Frequency bands 35-50 (high frequencies) are most important
         for distinguishing mines from rocks
