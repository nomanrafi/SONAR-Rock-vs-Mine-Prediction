                          TROUBLESHOOTING
================================================================================

Issue: "ModuleNotFoundError: No module named 'xgboost'"
Solution: Use Python 3.11 from conda (as in the launcher scripts)

Issue: Port 5000 already in use
Solution: 
  - Stop any other Flask apps running
  - Or modify app_sonar_predict.py, change:
    app.run(host='0.0.0.0', port=5000)
    to:
    app.run(host='0.0.0.0', port=5001)

Issue: "Models not loaded" message in browser
Solution:
  - Make sure you're running with Python 3.11
  - Check that all 4 .pkl files exist in the sonar project folder
  - Check console output for specific error messages

Issue: Can't find sonar_data.csv
Solution:
  - The web app doesn't need it at runtime
  - It's only used in SONAR_03.ipynb for training
  - All training data is already in the .pkl model files
