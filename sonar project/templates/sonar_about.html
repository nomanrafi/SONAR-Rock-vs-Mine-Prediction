üì° About SonarCheck

Machine Learning-Powered SONAR-Based Object Detection System

üéØ Project Overview

SonarCheck is an advanced machine learning application designed to classify underwater objects as either natural rocks or potential explosive mines using SONAR signal analysis. This system demonstrates the power of AI in maritime safety and naval applications.

Two Primary Goals

üéØ Goal 1: Classification

Accurately predict whether a detected underwater object is a rock or mine using 60 SONAR frequency bands.

üîç Goal 2: Interpretation

Identify which SONAR frequencies are most important in distinguishing rocks from mines.

üìä Dataset Information

Source

UCI Machine Learning Repository - SONAR Dataset

This dataset is used for binary classification of underwater objects and is widely recognized in the machine learning community.

Dataset Characteristics

  • Total Samples: 208 instances
  • Rocks: 111 samples
  • Mines: 97 samples
  • Features: 60 frequency bands (11.25 kHz to 100 kHz)
  • Value Range: 0.0 to 1.0 (normalized amplitude)

Equipment Used

  • SONAR Type: Goodman Tonals & Mirrorbird Standard SONAR
  • Application: Underwater object detection and mine detection
  • Frequency Range: 11.25 kHz to 100 kHz (60 bands total)

‚öôÔ∏è How It Works

1. SONAR Frequency Data Input

Users provide 60 normalized frequency band values (0.0 to 1.0) representing SONAR echo signals across different frequency bands.

2. Machine Learning Models

Primary Model: XGBoost Classifier

  • Ensemble gradient boosting method
  • Excellent for tabular data with complex patterns
  • Captures non-linear relationships in frequency data

Backup Model: Logistic Regression

  • Simple, interpretable linear model
  • Used if primary model fails
  • Provides statistical confidence

3. Classification Process

  1. Input frequency values are standardized (z-score normalization)
  2. XGBoost model predicts object class (Rock or Mine)
  3. Confidence probability is calculated (0-100%)
  4. Risk level is determined based on confidence
  5. Feature importance shows contributing frequency bands

4. Risk Assessment

The system provides risk levels and recommendations:

  • CRITICAL: High confidence mine detected - immediate evasion
  • HIGH: Probable mine - recommend caution
  • MODERATE: Uncertain - recommend investigation
  • SAFE: Likely rock - normal operations can continue

üåä SONAR Signal Patterns

ü™® Rock Signals

Natural rocks produce smooth, distributed SONAR reflections across frequency bands:

  • Smooth amplitude distribution
  • Gradual increase/decrease across frequencies
  • Low peak amplitudes
  • Natural decay patterns

üí£ Mine Signals

Manufactured mines produce concentrated SONAR signatures, especially in high frequencies:

  • Sharp peaks in specific frequency bands
  • Concentrated energy in high frequencies (bands 40-59)
  • Distinct resonance patterns
  • Artificial symmetry in signal structure
