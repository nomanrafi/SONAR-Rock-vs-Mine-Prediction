About The Project

Understanding the technology behind underwater object detection.

Project Overview

This project utilizes machine learning to classify underwater objects as either Rocks or Mines (Metal Cylinders) based on SONAR signals. The system analyzes the strength of SONAR returns across 60 different frequency bands to identify characteristic patterns.

How It Works

1
Data Collection

SONAR signals are bounced off underwater objects. The return signal strength is measured across 60 frequency bands.

2
Preprocessing

Signal data is normalized and processed to ensure consistency.

3
Classification

The trained XGBoost model analyzes the frequency pattern. Rocks tend to have smooth, distributed signals, while mines show concentrated energy in specific high-frequency bands.

4
Result

The system outputs the object type (Rock/Mine), a confidence score, and a risk assessment.
